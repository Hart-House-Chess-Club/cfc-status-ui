About CFC Rating Processor

Learn how to use this tool and format your CSV files correctly.

How It Works

  1. 1

    Upload your tournament CSV file

    The file should contain player information including CFC IDs

  2. 2

    Automatic data fetching

    The tool fetches current CFC ratings, membership status, and FIDE information for each player

  3. 3

    Ranking and output

    Players are ranked by rating and the updated file is available for download

CSV Format Requirements

Required Format:

Your CSV file should have at least 3 columns:

  • Column 0: Starting rank/position
  • Column 1: Player name
  • Column 2: CFC ID (default column)

Special CFC ID Values:
  • NA or N/A - Player without CFC membership
  • Empty cell - Will be skipped
  • Numeric ID - Will be looked up in CFC database

Output Format

The processed CSV file will include all original columns plus these new columns:

Column Description Example Values
CFC_Quick_Rating Current CFC quick rating 1850, 1654, Unrated
CFC_Member_Status Membership status Active, Expired, Not Found
CFC_Expiry_Date Membership expiry date 2025-12-31, N/A
FIDE_ID FIDE identification number 2600528, N/A
CFC_First_Name First name from CFC database John, Jane
CFC_Last_Name Last name from CFC database Smith, Doe

Configuration Options

Event Date

Used to determine if CFC memberships are current on the event date. If a membership expires before the event date, it will be marked as expired.

CFC ID Column Index

Specify which column contains the CFC IDs (0-based indexing). Default is column 2. Adjust this if your CFC IDs are in a different column.
